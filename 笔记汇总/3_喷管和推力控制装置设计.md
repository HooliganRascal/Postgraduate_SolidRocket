# 喷管和推力控制装置设计

## 概述
1. 喷管的作用：
    - 根据药柱燃面，控制喷管喉部面积大小保证燃烧室具有一定工作压强，使药柱正常燃烧
    - 药柱燃烧生成的燃气的热能转化为动能，产生推力
    - 改变推力方向

   固发喷管为非冷却结构或消融冷却结构，分为潜入/非潜入，固定/可动
2. 喷管设计：
    - 气动设计：确定内表面形状
    - 热防护设计：选择烧蚀和绝热材料，确定厚度结构
    - 结构设计：设计喷管支撑结构
3. 推力控制：
    - 包括推力向量控制和推力终止
    - 常用的有反喷管推力终止装置
4. 设计要求
    - 喷管结构要求：类型、结构尺寸
    - 喷管工作要求：最大压强和平均压强、推进剂类型
    - 性能要求：效率高、工作可靠、结构质量轻、推力向量控制结构合理、外壁面温度小于规定值、喉部烧蚀率低
    - 环境条件：工作环境温度、工作高度、贮存环境、振动过载
    - 可靠性和维修性要求：可靠性指标、维修性
5. 喷管设计程序：
    - 喷管设计
    - 气动、热防护、结构
    - 迭代设计、性能分析与结构分析
    - 结构件以及喷管冷试
    - 完成生产、试验
    - 分析、解剖、修正设计

## 喷管的气动设计
1. 收敛段设计：
    - 接头半径 $R$
    - 收敛半角 $\beta$
    - 入口段椭圆长半轴 $R_{a}$ 和短半轴 $R_{u}$
    - 入口段长度 $L_{i}$
2. 喉部设计：
    - 喉段长度 $L_{t}$
    - 喉部半径 $R_{t}$
    - 喉部曲率半径 $R_{c}$
    - 喉部下游半径 $R_{d}$
3. 扩散段设计：
    - 初始膨胀半角 $\alpha_{m}$ 与出口膨胀半角 $\alpha_{e}$（特型喷管的损失系数 $\eta = \eta(\alpha_{m},\alpha_{e})$ ）
        $$\eta = \frac{1}{2}\left[ 1+\cos\left( \frac{\alpha_{m}+\alpha_{e}}{2} \right) \right]$$
    - 膨胀比 $\varepsilon_{A}$

### 收敛段和喉部
1. 非潜入喷管：$\beta=45^o$，$R_{c}=(1\sim 2)R_{t}$
2. 潜入喷管：$R_{a}/R_{u}=3:2$，$R_{a}=(1\sim 2)R_{t}$，$L_{i}=(1\sim 2)R_{t}$
3. 长尾喷管：
    - 尾管内径 $d_{itn}=(1.3\sim 2.0)d_{t}$
    - 尾管入口处的速度系数 $\lambda_{l}\leq 0.4$
    - 圆筒段长度 $L_{tn}$ 和内径 $d_{itn}$ 之比存在上限
    - 其他设计类似

### 扩散段
1. 扩张比 $\varepsilon_{A}$ 和压强比 $\xi_{p}$ 的关系
    $$\varepsilon_{A}=\frac{A_{e}}{A_{t}}=\frac{ \displaystyle \sqrt k \left( \frac{2}{1+k} \right)^{\frac{k+1}{2(k-1)}}  }{\displaystyle \left( \frac{P_{e}}{P_{c}} \right)^{\frac{1}{k}}\sqrt{ \frac{2k}{k-1}\left[ 1-\left( \frac{P_{e}}{P_{c}} \right)^{\frac{k-1}{k}} \right] }  }$$
    
    总体不给扩张比就取最大比冲进行计算。

2. 锥形、特形（钟形，变分原理法和直接优化法）
    - 锥形扩散段：
        - 半锥角取 $\alpha = 15^o$
        - 扩张损失系数：
            $$\eta_{div}=\frac{1+\cos\alpha_{e}}{2}$$
          其中 $R_{tD}$ 为喉部下游半径，$\alpha_{e}$ 为出口膨胀半角
        - 出口内半径：$R_{e}=R_{t}\sqrt \varepsilon_{A}$
        - 扩散段长度：
            $$L_{e}=\frac{R_{e}-(R_{t}+R_{tD}-R_{tD}\cos\alpha_{e})}{\tan \alpha_{e}} + R_{tD}\sin\alpha_{e}$$
    - 特形扩散段
        - 特征线法（双圆弧法近似）
        - 饶氏方法（抛物线法近似）
        - 三次多项式优化曲线

## 喷管的热防护设计
1. 热防护工作：
   喷管的气动内型面绝热耐烧蚀，保证内型面连续、完整、控制外壁面温度
2. 热防护设计：
   使内型面不改变或者少改变，支撑结构的温度控制在允许的范围内
3. 主要任务：
    - 选择热防护材料（烧蚀和绝热）
    - 进行传热、烧蚀和热应力分析
    - 确热防护结构形状和尺寸

### 喷管的热环境
1. 对流、颗粒辐射、碰撞传热
2. 局部对流换热系数 $a_{c}$：巴兹公式，随后可计算热流密度
    $$q=a_{c}\sqrt{\frac{h_{r}}{h_{c}}}T_{r}=a_{c}(T_{r}-T_{w})$$
   其中 $h_{r}$ 为燃气恢复比焓，$h_{c}$ 为燃气静比焓，$T_{r}$ 为燃气恢复温度，$T_{w}$ 为壁面温度。
3. 辐射和碰撞传热用试验确定

### 热防护层厚度估算
1. 设允许烧蚀的厚度为 $\delta_{a}$，估算的烧蚀厚度为 $\delta_{e}$，则安全裕度为：
    $$f=\frac{\delta_{a}-\delta_{e}}{\delta_{e}}$$
2. 经验方法计算烧蚀和碳化厚度
    - 巴兹方法换算烧蚀率 $r_{a}$ ，然后可计算烧蚀厚度：
    $$\delta_{a}=r_{a}t$$
    - 碳化厚度 $\delta_{c}$ 由经验公式确定，$A$，$B$，$m$ 均为经验常数：
        $$\delta_{c}=At^{m}e^{\displaystyle -\frac{B}{q}}$$
      其中 $q$ 为热流密度
3. 考虑 $0.2\sim 0.5$ 的碳化安全裕度，得到热防护层总厚度：
    $$\delta = (1.2\sim 1.5)(\delta_{a}+\delta_{c})$$

### 热防护材料
1. （耐）烧蚀材料：
    - 石墨、难熔金属、碳/碳复合材料、陶瓷/陶瓷复合材料
    - 用于喉衬、收敛段和扩散段的烧蚀层
2. 绝热材料
    - 喉衬的背壁、烧蚀层背壁的绝热层。
3. 烧蚀热效率：
    - 一定背壁温升时总热量和材料初始质量 $m_{m}$ 之比：
        $$E_{ef}=\frac{q_{cv}t_{a}}{m_{m}}$$
      其中 $t_{a}$ 为工作时间。
    - 选收敛段和扩散段材料
4. 有效烧蚀热：
    - 总热量与材料损耗量之比：
        $$H_{ef}=\frac{q_{cv}t_{a}}{\Delta m_{m}}$$
    - 选喉衬材料
5. 材料选择：
    - 喉衬：
        - 工作时间较短、喉径较小，可采用高强高密度石墨
        - 工作时间相当，喉径相当，可采用石墨渗铜
        - 严格控制喉径，壁厚很薄，可采用碳/碳复合材料
        - 工作时间较长，喉径较大，可采用毡基碳/碳复合材料或多向编织碳/碳复合材料
        - 几种材料：多晶石墨、热解石墨、碳/碳复合材料、难熔金属、陶瓷和陶瓷复合材料
    - 收敛段和扩散段：
        - 工作时间较短，燃气流速低，结构质量要求不严格，收敛段绝热层采用模压高硅氧/酚醛材料结构
        - 工作时间较长，收敛段绝热层采用高硅氧/酚醛复合模压或高硅氧布/酚醛和碳布/酚醛复合缠绕材料结构
        - 工作时间较短，扩张比较小，扩散段绝热层采用模压高硅氧/酚醛材料结构、酚醛或高硅氧布/酚醛缠绕材料结构
        - 工作时间较短，扩张比较大，扩散段绝热层采用高硅氧布、酚醛和碳布/酚醛复合缠绕材料结构
            - 内层碳/酚，外层高/酚均用平行
            - 内层碳/酚用斜布带（$10^o\sim 20^o$），外层高/酚用平行
        - 常用材料：
            - 石棉-酚醛树脂（只有模压制品，作喉衬背壁和绝热层，不作烧蚀层）
            - 高硅氧-酚醛树脂（可作烧蚀层也可作绝热层）
            - 碳-酚醛树脂（可作烧蚀层也可作绝热层，烧蚀性能较好）
        - 热防护层和壳体用胶黏连接，常温固化胶粘接的剪切强度高。
    - 尾管材料：
        - 流量小，时间短，推进剂不含铝粉，用模压高硅氧/酚醛材料结构、模压碳/酚醛和高硅氧/酚醛复合材料结构或碳布/酚醛和高硅氧布/酚醛层压结构
        - 流量大，时间长，用镶嵌石墨，碳布/酚醛和高硅氧布/酚醛复合缠绕结构
            - 镶嵌材料：高强度高密度石墨、高强度石墨渗铜等
            - 镶嵌长度：入口一段/整个尾管镶嵌，长厚比小于 $3\sim 4$
            - 直筒段内层用斜布带缠绕材料结构缠绕碳布/酚醛，外层用高硅氧布/酚醛平行缠绕材料结构

## 喷管的结构设计

### 支撑结构
1. 承受载荷
2. 收敛段圆筒段厚度 $\delta$ 可用第一、第四强度理论确定
3. 喉部可设计凸台或加装止动螺钉
4. 扩散段厚度可以设计相对薄一点，出口可以加挡环或者在结构件与扩散段绝热层之间加止动螺钉


### 连接结构
1. 法兰、螺栓、螺纹、卡环连接
2. 结构尺寸无限制，常采用法兰-螺栓连接结构，经验公式确定法兰厚度
2. 结构尺寸有限制，常采用螺纹连接结构

### 密封结构
1. 固定喷管常采用 $O$ 形橡胶密封圈进行端面密封
2. 侧面密封结构，横截面积与压缩量相比端面要小一些，可参照相关公式进行计算

### 喷管堵盖结构
1. 金属纯铝堵盖
2. 非金属夹布（丝绸）橡胶软堵盖

### 粘接剂和密封剂
1. 粘接剂：粘接零部件
2. 密封剂：引导燃气通过零件间隙

### 热防护设计
1. 喉衬结构
    - 结构设计
    - 烧蚀率估算
    - 碳/碳喉衬烧蚀计算
2. 烧蚀层和绝热层的厚度
    - 碳化层厚度和剥蚀厚度
    - 烧蚀率
    - 烧蚀层和绝热层厚度
    - 预估烧蚀厚度
    - 安全裕度
    - 绝热层设计总厚度

## 推力向量控制装置的要求和类型
1. 按预定弹道飞行
2. 纠正姿态
3. 变轨

### 推力向量控制装置的基本要求
1. 最大侧向力
2. 作动力矩要小
3. 动态性能要好
4. 推力损失应小
5. 工作可靠、质量小、结构紧凑、维护使用方便、易于制造、成本低廉

### 推力向量控制装置分类
1. 喷管活动：固定喷管和可动喷管
2. 工作原理和相应伺服或供应系统：
    - 机械类：固定、可动、辅助发动机
    - 流体喷射类：液体二次喷射（惰性/活性）、气体二次喷射（空气/燃气）

### 推力向量控制装置的类型选择
1. 致偏能力：最大侧向力或最大偏斜角
2. 频率响应：推力向量控制装置对控制信号响应的快慢
3. 伺服机构的功率和尺寸：作动力矩大小
4. 轴向推力的影响：除了流体二次喷射是增推力，其余都是减推力

## 摆动喷管

### 单轴摆动喷管
1. 结构特点：
    - 铰链喷管，一个方向的控制力，四喷管布置方案（$+,\times$）
    - 固定体：金属法兰，喷管座，凹球台
    - 活动体：$O$ 形密封圈，凸球台
    - 铰链接头：转轴、轴承、凹球台、凸球台、$O$ 形密封圈
    - 亚声速分离线、超声速分离线、潜入式摆动喷管（气流滞止区分离线）
2. 关键零部件的选择
    - 轴承：体积小，承载能力大，自动调心，转动灵活，摩擦力小（关节轴承）
    - $O$ 形密封圈

### 柔性喷管
1. 基本结构和工作原理
    - 全轴摆动，轴对称，有作动器
2. 柔性接头的结构参数
3. 柔性喷管的侧向力和作动力矩
4. 柔性喷管的特点及应用：
    - 无分离线、偏转能力大、推力损失小
    - 密封性好、响应快、可靠性高

### 液浮喷管
1. 工作原理：液浮轴承连接
2. 基本结构形式：液浮囊——柱形，锥形，球形（局部、整体）
3. 优点：
    - 作动力矩小
    - 液浮轴承既是支撑件又是密封件，热防护简单
4. 缺点：
    - 摆心飘移较大
    - 囊内液体易渗漏

### 珠承喷管
多为潜入式喷管，分离线在气流滞止区

## 流体二次喷射装置
1. 在扩散段某一位置向喷管中的主流横向注入二次流来控制所需侧向力
2. 频率响应快、效率高、喷管结构简单（固定）和比冲损失小

### 液体二次喷射
1. 液体工质可以在环境温度下贮存和使用，管路和喷射器不受高温燃气烧蚀
2. 液体工质密度大，便于贮存
3. 侧向力有限，装置结构复杂，勤务处理不方便

### 气体二次喷射
1. 侧向比冲高于液体
2. 燃气二次喷射装置
    - 取自燃烧室的高温燃气，零部件烧，质量小
    - 管路和阀门需要承受高温燃气烧蚀，燃气中的氧化铝沉积可能堵死，引出的燃气和燃烧室压强双向影响
3. 低温燃气二次喷射装置：专门的燃气发生器，效率低一些
4. 空气二次喷射装置：取自周围的空气，装置简单，质量小，只适用于大气层
5. 气体二次喷射装置尚未用于正式型号的发动机

## 推力终止装置设计

通常使用反喷管推力终止装置

### 反喷管推力终止装置设计要求

1. 接到指令后及时解锁
2. 轴向反推力必须瞬时建立，得到负推力后持续一段时间
3. 解锁和打开的同步性要好
4. 结构简单、易于加工、可靠性高、安全性好

### 确定反喷管的喉径：
1. 燃烧室平衡压强
2. 斜切口（出口端面被切好，使导弹具有良好的气动外形）喷管推力系数
3. 反喷管的打开机构
    - 滚珠锁紧爆炸螺栓打开机构：结构简单，动作迅速，可靠
    - 聚能切割打开机构：质量小，打开时间短，同步性好，安全可靠

## 喷管分离流动计算

### 喷管工作状态
假定条件：
1. 一维定常
2. 闭口系，不计摩擦
3. 气体化学成分、气体常数不变
四种工作状态：
1. 欠膨胀和完全膨胀 $P_{b}\leq P_{1}$ （到完全膨胀）
2. 喷管出口外激波 $P_{1} < P_{b}\leq P_{2}$ （斜激波到正激波）
3. 喷管内部激波 $P_{2} < P_{b}\leq P_{3}$ （出口激波到喉部激波）
4. 管内全亚声速流 $P_{b} > P_{3}$

### 流动分离状态
1. 边界层分离：粘性，逆压梯度，扩压减速
2. 边界层与激波的相互作用
    - 层流入射激波，边界层变厚，形成反射激波
    - 激波造成的逆压梯度梯度变大，发生分离，分离区自由边界反射为膨胀波
    - 波后气体折回壁面形成湍流边界层重新贴回壁面，第二道反射激波
3. 喷管分离现象
    - 分离流动：激波进入喷管内部同时伴随着边界层的分离
    - 自由激波分离：气流分离则气流不再附着于壁面上
    - 受限激波分离：气流分离后又附着于壁面上

### 分离点参数计算
1. 理论计算：
    - 正激波气流分离面
    - 非正激波气流分离面
2. 经验公式：
    - Summerfield 准则（出口壁面压力小于 $40\%$ 环境压力就会观察到气流分离）
    - Lawrence 采用稳态压力 $P_{p}$ 建立分离准则
    - Schmucker提出一种新的分离准则
    - 影响分离流动的因素：喷管型面、压强比、气流参数、壁面冷却、表面粗糙度等
